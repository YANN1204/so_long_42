So_long : Graphical 2D game
===========================

<br/>

Library used : Minilibx_Linux (Library 42's school)

<br/>

Step to launch :

1. ```make```

2. ```./so_long map_1.ber```

<br/>

Three type of map :
- map_1.ber (valid map)
- map_2.ber (invalid map)
- map_3.ber (extended map)

<br/>

Movement's player :

- Key 'W' -> Up
- Key 'S' -> Down
- Key 'A' -> Right
- Key 'D' -> Left

Endgame condition -> When all collectibles are collected.
